Electrical Wiring and Design

During the second semester of my Sophmore Year I started working with the Jet Engine branch of the WPI AIAA club. I designed the first version of the control box for the engine and am currently working with the club to revamp and improve the control systems to be both safer and more reliable. Along with this I did the inital Estop and sensor wiring for the engine, along with debugging when errors came up during testing.

New Control System

My current proposed design will have 3 seperate safety mechanisms, two soft stops and one hard Emergency Stop. The control signal is generated from an Arduino which will allow us to replace the FlySki RC flight controller we are currently using. I also intend to look into the data termainal that we currently use for RPM reading from the engine and attempt to replace the terminal with the Arduino for real time display of engine data on the control box.

Data Collection

The next step in the elctronics and sensor design for the engine is to develop our data collection with a Raspberry Pi Zero on the test stand. This will be used to read engine RPM, load cells and pitot tubes to record and display that data in real time. I am also considering the possiblity in having it host a webserver for remote viewing of the data.
